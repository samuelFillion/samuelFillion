Samuel Fillion

J'ai 21 ans, et j'ai terminé ma formation en intégration multimédia au printemps 2018. J'ai toujours aimé le côté artisitique qu'offrait le développement web. J'ai de très bonnes connaissances de la suite Adobe(Photoshop, Illustrator, After Effects, Premiere Pro), pour la création d'interfaces utilisateurs intuitives. J'ai également de bonnes connaissances en programmation (Python, PHP, JavaScript).

Je souhaite maintenant m'orienter vers la création de jeux vidéo. Les jeux vidéo, comme une grande majorité de gens de ma génération, ont une place assez importante dans ma vie. Que ce soit les souvenirs d'enfance ou encore les compétitions e-sport, de plus en plus populaire! C'est un domaine stimulant, ludique où chacun peut trouver son petit bonheur.

Mes intentions

Programmation d'un jeu de tir 2D

Ce projet a été réalisé d'A à Z dans un cadre scolaire. Il s'agit d'un jeu de tir.

Pour ce projet, j'ai utilisé la programmation orientée-objet.

Chacune des images a d'abord été faite avec l'aide d'Adobe Illustrator, image par image, puis animé dans Adobe Animate.

Le jeu utilise un tri sur l'axe des x(pour éviter les obstacles et les ennemis), un défilement en parallax pour le décor, un compteur de point et 3 niveaux.

Technologies : Adobe Animate, Adobe Illustrator, JavaScript(avec TypeScript et Create.js)

Programmation d'un jeu d'Othello

Ce projet a été réalisé en équipe de 2 dans un cadre scolaire.

Nous avons programmé chacun 2 classes d'envergure assez égale.

Nous avons utilisé la programmation orientée-objet en Python.

C'est un jeu de stratégie qui se joue à 2 joueurs. Pour jouer un tour, les joueurs doivent manger les pièces de l'adversaire, sinon le tour est passé. Après deux tours passés, le joueur a perdu. L'autre moyen de gagner est de remplir la planche avec le plus de pions de sa couleur. Il est possible de sauvegarder une partie et évidemment de charger une partie. Il est également possible de faire jouer un ordinateur.

Technologie : Python, GitHub

Court métrage - Ben Drowned

Ce projet a été réalisé en équipe de 2 dans un cadre scolaire.

Le travail a été partagé de manière égale pour ce qui est du tournage, du montage et des effets spéciaux.

Programmation et design d'un quiz (Progressive Web App)

Il s'agit d'un petit quiz portant sur la série «Stranger Things», une de mes séries préférées qui rappelle l'univers des blockbusters des années 80.

Ce projet était un prétexte pour explorer les applications web progressives

J'ai d'abord élaboré la maquette. J'ai ensuite traité les images, programmé les questions ainsi que les services workers pour que l'application soit fonctionnelle tout en étant hors-ligne. L'application est responsive et fonctionne donc sur ordinateur et téléphone, où elle prend les propriétés d'une application mobile.

Technologies : Sketch, JQuery, PHP, HTML, CSS, Service Workers
