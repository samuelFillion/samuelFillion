Programmation d'un jeu de tir 2D

Ce projet a été réalisé de A à Z dans un cadre scolaire. Il s'agit d'un jeu de tir.

Pour ce projet, j'ai utilisé la programmation orienté-objet.

Chacune des images ont d'abord été fait avec l'aide d'Adobe Illustrator, image par image, puis animé dans Adobe Animate.

Le jeu utilise un tri sur l'axe des x(pour éviter les obstacles et les ennemis), un défilement en parallax pour le décor, un compteur de point et 3 niveaux.

Technologies : Adobe Animate, Adobe Illustrator, JavaScript(avec TypeScript et Create.js)

Programmation d'un jeu d'Othello

Ce projet a été réalisé en équipe de 2 dans un cadre scolaire.

Nous avons programmé chacun 2 classes d'envergure assez égale.

Nous avons utilisé la programmation orienté-objet en Python.

C'est un jeu de stratégie qui se joue à 2 joueurs. Pour jouer un tour, les joueurs doivent manger les pièces de l'adversaire, sinon le tour est passé. Après deux tours passé, le joueur a perdu. L'autre moyen de gagner est de remplir la planche avec le plus de pions de sa couleur.

Il est possible de sauvegardé une partie et évidemment de charger une partie. Il est également possible de faire jouer un ordinateur.

Technologie : Python

Programmation et design d'un quiz (Progressive Web App)

Il s'agit d'un petit quiz portant sur la série «Stranger Things», une de mes séries préférées qui rappele l'univers des blockbusters des années 80.

Ce projet était un prétexte pour explorer les applications web progressives

J'ai d'abord élaboré la maquette. J'ai ensuite traité les images, programmé les questions ainsi que les services workers pour que l'application soit fonctionnelle tout en étant hors-ligne. L'application est responsive et fonctionne donc sur ordinateur et téléphone, où elle prend les propriétés d'une application mobile

Technologies : Sketch, JQuery, PHP, HTML, CSS, Services Workers
